# 一、ベルマン方程の始まり

## 1️⃣ Q関数の定義

$$
Q^\pi(s,a) = \mathbb{E}_\pi[G_t \mid s_t=s, a_t=a]
$$

これは以下を意味します：

> 状態sでアクションaを行い、その後戦略πに従って行動した場合の長期的な期待収益。

---

## 2️⃣ ベルマン最適方程式（重要）

$$
Q^*(s,a) = \mathbb{E}[r + \gamma \max_{a'} Q^*(s',a') \mid s,a]
$$

これは以下を教えてくれます：

> 現在のアクションの価値 = 現在の報酬 + 折引後の「将来の最適価値」

これは再帰的な関係です。

---

# 二、TD誤差とは何か？

ベルマン方程式を直接解くことはできません。

しかし、交互作用を通じてそれに近づくことができます。

定義：

$$
\delta = r + \gamma \max Q(s',a') - Q(s,a)
$$

これは以下を意味します：

> 新しい情報 − 以前の推定値

もしδ = 0であれば、ベルマン方程式を満たしていることになります。

---

# 三、Q学習の核心的な考え方

Q学習の更新規則は以下の通りです：

$$
Q(s,a) \leftarrow Q(s,a) + \alpha \delta
$$

詳細に展開すると：

$$
Q(s,a) \leftarrow Q(s,a) + \alpha \left( r + \gamma \max Q(s',a') - Q(s,a) \right)
$$

これは以下の意味です：

1. 一歩進んで観察する。
2. TD誤差を計算する。
3. その誤差を使ってQ値を修正する。
4. 收束するまでこれを繰り返す。

---

# 四、実際の例：3状態の迷路

簡単な環境を構築してみましょう。

---

## 🌱 環境設定

状態：

S1 → S2 → S3（終点）

アクション：

* 右（→）
* 左（←）

報酬：

* S3に到達した場合 +10
* その他の各ステップでは -1

割引係数：$\gamma = 0.9$

学習率：$\alpha = 0.5$

---

## Q表の初期化

初期のQ表はすべて0です：

| 状態 | 左 | 右 |
| -- | - | - |
| S1 | 0 | 0 |
| S2 | 0 | 0 |

---

## 交互作用による更新を開始する

### 第1回の交互作用

S1から右に進んでS2に到達。

報酬 = -1

現在のQ(S1,右) = 0

S2での現在の最大Q値も0

目標値：$-1 + 0.9×0 = -1$

TD誤差：$-1 - 0 = -1$

更新後：$0 + 0.5×(-1) = -0.5$

---

更新後のQ表：

| 状態 | 左 | 右 |
| -- | - | ---- |
| S1 | 0 | -0.5 |
| S2 | 0 | 0 |

---

### 第2回の交互作用

S2から右に進んでS3に到達。

報酬 = +10

S3は終点なので、将来のQ値は0

目標値：$10 + 0.9×0 = 10$

TD誤差：$10 - 0 = 10$

更新後：$0 + 0.5×10 = 5$

---

更新後のQ表：

| 状態 | 左 | 右 |
| -- | - | ---- |
| S1 | 0 | -0.5 |
| S2 | 0 | 5 |

---

### 第3回の交互作用

再びS1から右に進んでS2に到達。

報酬 = -1

この時、S2での最大Q値は5になっている

目標値：$-1 + 0.9×5 = 3.5$

TD誤差：$3.5 - (-0.5) = 4$

更新後：$-0.5 + 0.5×4 = 1.5$

---

更新後のQ表：

| 状態 | 左 | 右 |
| -- | - | --- |
| S1 | 0 | 1.5 |
| S2 | 0 | 5 |

---

## 何が起こったのか？

以下のようなことが起こりました：

* S2の価値が最初に上昇した。
* その後、S1の価値が「伝わって」きた。

これを以下と呼びます：

> 価値の逆伝播（reward propagation）

強化学習はまさにこのように機能します：

> 報酬が終点から前方に伝わっていく。

---

## 継続的な更新後の傾向

最終的に収束した時：

S2の右方向の価値 ≈ 10
S1の右方向の価値 ≈ 8

なぜなら、$-1 + 0.9×10 = 8$だからです。

これがベルマン方程式の解です。

---

## 最適な戦略をどのように得るか？

最終的には以下のようになります：

$$
\pi(s) = \arg\max_a Q(s,a)
$$

つまり：

* S1では右を選ぶ。
* S2でも右を選ぶ。

---

## 総括

Q学習は以下の3つのことを行います：

1. ベルマン方程式を目標とする。
2. TD誤差を使って偏差を測定する。
3. 小さなステップで更新を行い、最適なQ値に近づける。

本質的には：

> サンプルを使ってベルマン最適方程式を近似して解くというものです。
