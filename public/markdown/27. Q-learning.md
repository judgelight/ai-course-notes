---

# 一、贝尔曼方程开始

## 1️⃣ Q 函数定义

$$
Q^\pi(s,a) = \mathbb{E}_\pi[G_t \mid s_t=s, a_t=a]
$$

表示：

> 在状态 s 采取动作 a，然后按策略 π 行动的长期期望回报。

---

## 2️⃣ 贝尔曼最优方程（重点）

$$
Q^*(s,a) = \mathbb{E}[r + \gamma \max_{a'} Q^*(s',a') \mid s,a]
$$

它告诉我们：

> 当前动作的价值 = 当前奖励 + 折扣后的“未来最优价值”

这是一种递归关系。

---

# 二、TD 误差是什么？

我们无法直接求解贝尔曼方程。

但可以通过交互来逼近它。

定义：

$$
\delta = r + \gamma \max Q(s',a') - Q(s,a)
$$

它表示：

> 新证据 − 旧估计

如果 δ = 0，说明我们满足贝尔曼方程。

---

# 三、Q-learning 的核心思想

Q-learning 的更新规则：

$$
Q(s,a) \leftarrow Q(s,a) + \alpha \delta
$$

展开就是：

$$
Q(s,a) \leftarrow Q(s,a) + \alpha \left( r + \gamma \max Q(s',a') - Q(s,a) \right)
$$

它的含义是：

1. 观察一步
2. 计算 TD 误差
3. 用误差修正 Q 值
4. 重复直到收敛

---

# 四、实际例子：三状态迷宫

我们构造一个简单环境。

---

## 🌱 环境设定

状态：

S1 → S2 → S3（终点）

动作：

* 右（→）
* 左（←）

奖励：

* 到达 S3 得 +10
* 其他每步 -1

折扣因子：$\gamma = 0.9$

学习率：$\alpha = 0.5$

---

## 初始化 Q 表

初始 Q 表全为 0：

| 状态 | 左 | 右 |
| -- | - | - |
| S1 | 0 | 0 |
| S2 | 0 | 0 |

---

## 开始交互更新

### 第一次交互

在 S1 选择 右 → 到 S2

奖励 = -1

当前 Q(S1,右)=0

S2 当前最大Q=0

计算目标：$-1 + 0.9×0 = -1$

TD误差：$-1 - 0 = -1$

更新：$0 + 0.5×(-1) = -0.5$

---

更新后：

| 状态 | 左 | 右    |
| -- | - | ---- |
| S1 | 0 | -0.5 |
| S2 | 0 | 0    |

---

### 第二次交互

在 S2 选择 右 → 到 S3

奖励 = +10

S3 是终点，未来Q=0

目标：$10 + 0.9×0 = 10$

TD误差：$10 - 0 = 10$

更新：$0 + 0.5×10 = 5$

---

更新后：

| 状态 | 左 | 右    |
| -- | - | ---- |
| S1 | 0 | -0.5 |
| S2 | 0 | 5    |

---

### 第三次交互

再次从 S1 → 右 → S2

奖励 = -1

现在 S2 最大Q=5

目标：$-1 + 0.9×5 = 3.5$

TD误差：$3.5 - (-0.5) = 4$

更新：$-0.5 + 0.5×4 = 1.5$

---
更新后：

| 状态 | 左 | 右   |
| -- | - | --- |
| S1 | 0 | 1.5 |
| S2 | 0 | 5   |

---

## 发生了什么？

你看到：

* S2 的价值先变大
* 然后 S1 的价值被“传递”回来

这叫：

> 价值反向传播（reward propagation）

强化学习就是这样：

> 奖励从终点往前传播

---

## 持续更新后的趋势

最终收敛时：

S2 右 ≈ 10
S1 右 ≈ 8

因为：$-1 + 0.9×10 = 8$

这就是贝尔曼方程的解。

---

## 如何得到最优策略？

最终：
$$
\pi(s) = \arg\max_a Q(s,a)
$$

也就是：

* S1 选 右
* S2 选 右

---

## 总结

Q-learning 做了三件事：

1. 用贝尔曼方程作为目标
2. 用 TD 误差衡量偏差
3. 用小步更新逼近最优Q

本质是：

> 用样本近似求解贝尔曼最优方程
